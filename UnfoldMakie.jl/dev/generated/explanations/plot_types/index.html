    The Dilemma of Multidimensionality

    Note

    Please read the paper The Art of Brainwaves, if you want to know how we come up with these plot types.

    EEG – multidimensional data and could be presented differently.

    Possible dimensions:

    • Voltage (must have)
    • Time
    • Number of channels (1-128)
    • Spatial layout of channels
    • Experimental conditions
    • Trials/subjects

    Each way of ERP presentation is a choice of dimensions. Hard to show meaningfully more than 3 dimensions.

    Plot types

    Each plot type can represent several dimensions. Here we represented 8 plot types.
